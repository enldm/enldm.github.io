青龙面板刷京豆

本文以docker部署为教程，其余教程参考 https://hub.docker.com/r/whyour/qinglong

一：拉取镜像

拉取whyour的镜像 

docker pull whyour/qinglong

二：部署镜像

docker run -dit   -v $PWD/ql:/ql/data   -p 5700:5700   --name qinglong   --hostname qinglong   --restart unless-stopped   whyour/qinglong:latest

三：启动Web管理

打开 ip:5700 ，按照步骤完成设置。（推荐打开两步验证并关闭京东免密支付）

四：定时任务

定时任务点击新增，复制以下内容（faker库tg第一时间通知频道 https://t.me/scriptalking ）

二选一即可

Faker2 助力池版：

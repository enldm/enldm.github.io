Screen

参考文章:

  • https://blog.csdn.net/carefree2005/article/details/122415714

使用

apt update
apt install screen



screen -S Window1    //创建窗口


screen -d    //或者使用ctrl+a,然后输入d


exit         //结束窗口


screen -r id   //重新连接会话

Mount

在/dev下的设备不能直接访问，需要挂载到别的目录(一般是/mnt下)

新建一个目录可供访问

mkdir /mnt/disk0

然后选择你需要挂载的分区，可以通过 df -h查看剩余大小来判断

mount /dev/sda1 /mnt/disk0

如果想取消挂载

umount /dev/sda1

也可以修改/etc/fstab来实现开机自动挂载。

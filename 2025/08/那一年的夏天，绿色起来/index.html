那一年的夏天，绿色起来

虽然开始时间没有精确到是具体的某一分钟，但似乎大家都已就位。我带着萌新（Id:EnonrGennki）正前往菊花正中心的孙逸仙雕像，准备把 CF 直接毒掉。此时我俩身上 160+的 key，能不能把春熙路连完以及去肉防，还是个未知数。8 月 30 这天成都正好大降温，又赶上这个季度的 Field 活动，可谓是出门 walk 的好时机。

时间回到六月底，不知道是五套还是饺子提出扎针包——一个对我来说多么遥远的词汇啊。上次成都绿军扎针包还是 2017 年的达芬奇首级【我为鸽狂针包赛】，那天似乎在补课没参加，所以到现在我一次针包都没参与过。相比于 XMA 这种对抗性质的活动，针包的强度应该要低一点，如果没有敌军来捣乱的话。

key 都还没开始摸，就有人扰乱军心，严查！

要致富，先种树。要扎包，先摸 key。而胖次作为麻将的忠诚拥护者，坚持发扬成绿搓麻的精神。顺带提醒，下方有多张约饭图，还在熬夜的小朋友请熬到天亮再观看。

7 月 5 号，在饭后开展了第一次摸 Key 活动！美团这天发了一大堆到店 0 元购的奶茶卷，加上进肚子的当天一共白嫖了四杯水…

一周后散打老板来蓉暨漂流箱首发。关于约饭和漂流箱这里就不再重复了（其实那天下午到晚上去看成都邦邦的同人 live 去了，没来）。胖次当天就已经摸到了 150key，不愧是 hack 仙人。又是一周后，没见到漂流箱的我赶紧到春熙路约了一波天妇罗。次日周日在大川巷打完麻将造披萨。

25Q3 活动下一阶段+Delta Campaign: Fields 也将在 30 号当天开始，如果当天有游客在大规模拉 CF 将会对我们的计划造成影响，不过当务之急还是多摸 key（开饭）

活动前两周 Intel 截图中，machina 已经掉了不少，绿军也造了不少 block

23 号，活动开始前的最后一个周末。原本这天我是打算当特种兵去上海看邦高祖的 live 的，无奈根本抢不到票，遗憾退场。手上的 key 似乎还是不够，当天晚上带着伞过来再摸一波，喜提 key+26。AI 太好用了你们知道吗

有一说一茶白道的茉莉绿茶真难喝。 在成都人民经历了四十度高温天气的烤验后，接下来的每一天都可谓是凉风习习啊。

正在高高兴兴的消耗手里现有的垃圾 key，走之前玉林蓝军 ChenBW 给我来了句多插点 jio 我好去炸，还好只是想刷刷炸脚牌子。

活动头天，经过投票决定活动于周六下午开始。成都正好又来了波大降温，白天最低气温只有 19°C，让身在上海的 OP 背时不由得满身大汗。

周五晚上，楠哥在周围清理 glock，我支援两桶脚后也去上了增益 mod 以便胖次再薅点 Key。周六又正好约了大学同学玩，从早到晚都在外面，强度下来感觉不亚于去澳门打 XMA（我是不可能告诉你我澳门还没写，啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊）

十一点从天府红的 animate cafe 出来，原本想带两位萌新一起来参加的，其中一位稍微熟悉点游戏操作的德阳友军家中有事遗憾退场。这位 ORNA 大神天赋异禀，短短半天就能搞懂 Link、Glyph Hack，在当今年代缺少新人的情况下可真是罕见啊。最后一次摸脚+摸 key，事实证明这俩物资完全不够。

保持静默，语音里特工路陆续就位，而我还没走回 po 正中心下毒。我带着 EnonrGennki 穿过春熙路的人流快步前进，拿出两个 ADA 对着绿 Po 痛下杀手，快速丢炸打掉之后上好 4 个 A 盾。14:30 一到，语音里一声令下，活动在此刻开始！

我负责的区域是春熙路——太古里一带，考虑到人流太多+路线不熟，让 EnonrGennki 先去 IFS 楼里再走到大慈寺，我直接从春熙路去东大街外围到太古里。刚走出没两步，绿色的 link 从四面八方连进来。二十来度的气温在快步走+注意力高度集中的情况下也不算凉快了。（背时：已热死，勿念）

短短半小时，link 数量已经达到三百：

一小时左右，RJ51 大佬消耗完 key，部分人员手中的 key 和脚都开始出现不足。毒免疫时间也快过去了。此时语音里人员正在 OP 的指挥下进行增援、补充物资。

再说我这边，我来的时候身上有 1200 个脚+走路桶里压满了 1 脚，给 EnonrGennki 塞了一半之后，身上大概还有七八百。（为什么脚和 key 都不够？好好反思反思）除开大慈寺交给 EnonrGennki 搞完剩下的部分，兰桂坊那边的饺子物资出现不足，我赶紧扫了个美团单车到水井坊补了点 link 去和饺子汇合。拿了二十把 key 之后，我又穿过廊桥，沿着河边酒吧一路 link。没走几步路我的 key 也消耗完了，语音问了下玉林的蓝军在一段时间后仍然没有出现 log，我怕萌新防守不住，又赶紧骑车回 po 中心肉防。

稍微松一口气，消失的蓝军并没有奔着春熙路来，而是去了文殊院打掉了几根 link，这对我来说压力似乎小了点。注意到地图上春熙路锦华馆还有几根 link 没补，而且离得不远，我从 EnonrGennki 手上拿了一些 key 继续去补。

15:48，TG 弹了条消息，蓝军 Leexiaoyu 突然问了一句“你们完事了没 完事了我可就去了”。还有人手上还有几十把 key 没连，考虑到蓝军过来可能会直接毒掉，我先说了句再连会。

四点过再往后，人和物资都坚持不住了，在零零星星进来几根 link 之后，背时决定活动结束。经过计算，在不吃 apex 的情况下，摧毁一千多根 link 可获得 20w 的 ap，遗憾的是居然爆 AP 的时候不截图！

虽然打掉了一些 link，经过统计，最后 1204in+8out，合计是 1212 根 link，在成都已经算非常不错的成绩了！（如果川大望江+望江公园+玉林都连上就更多了，希望下次再破纪录）
